# AI4Edu

## Introduction
The system of AI in education, effectively incorporating ChatGPT and other AI models by Laravel.


## Packages

- Backend - Laravel
- Frontend - Vite & Inertia & React
- Authentication - Breeze
- Redis Queues - Horizon
- Log - Telescope
- UI Test - Dusk
- A/B Test - Pennant
- BDD - Behat
